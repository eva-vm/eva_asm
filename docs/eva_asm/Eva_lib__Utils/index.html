Module Eva_lib__Utils

This module contains utils for safe integer conversions.

exception Conversion_Error of string

Conversion Error exception. Raised by cst_of_string, adr_of_string and reg_of_string

val valid_cst : int -> [> `Cst of int ]

valid_cst i ensures that integer i belongs to the range of Eva consants. Return `Cst i if the test succeed, else raise Conversion_Error.

val valid_adr : int -> [> `Adr of int ]

valid_adr i ensures that integer i belongs to the range of Eva adresses. Return `Adr i if the test succeed, else raise Conversion_Error.

val valid_reg : int -> [> `Reg of int ]

valid_reg r ensures that integer r belongs to the range of Eva registers. Return `Reg r if the test succeed, else raise Conversion_Error.
